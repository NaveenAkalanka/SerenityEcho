## ğŸš€ Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/NaveenAkalanka/SerenityEcho.git
    cd SerenityEcho
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open locally**
    Visit `http://localhost:5173` in your browser.
